# New note in Single page app diagram
```mermaid
    sequenceDiagram
        actor user
        participant browser
        participant server

        user->>browser: SUBMIT "New note"
        browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
        activate server
        server-->>browser: [{ "content": "New note", "date": "2023-1-1" }]
        deactivate server

        Note right of browser: When the response is receive the JavaScript file renders the new note, does not make another request or redirects and reloads the page
```
